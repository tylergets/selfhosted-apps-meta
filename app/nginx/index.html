nginx

[{{ project_name|capitalize }}]({{ project_url }}) is a simple webserver with php support. The config files reside in `/config` for easy user customization.

docker
docs
Setup

Add your web files to /config/www for hosting.
Modify the nginx, php and site config files under /config as needed
Protip: This container is best combined with a sql server, e.g. mariadb

Docker Compose
# nginx - Generated by SH-Apps-Meta

version: "3.8"
services:
  nginx:
    image: linuxserver/nginx
    container_name: nginx
    restart: unless-stopped
    ports:
      #  http
      - 80:80
      #  https
      - 443:443
    volumes:
      #  Contains your www content and all relevant configuration files.
      - < /path/to/your/config>:/config
    environment:
      #  User ID
      - PUID=100
      #  Group ID
      - PGID=100
      #  Specify a timezone to use EG Europe/London
      - TZ=Europe/London
