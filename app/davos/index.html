davos

[{{ project_name|capitalize }}]({{ project_url }}) is an FTP automation tool that periodically scans given host locations for new files. It can be configured for various purposes, including listening for specific files to appear in the host location, ready for it to download and then move, if required. It also supports completion notifications as well as downstream API calls, to further the workflow.

docs
Setup

The application does not require any set up other than starting the docker container. Further documentation can be found on the davos GitHub repository page.

Docker Compose
# davos - Generated by SH-Apps-Meta

version: "3.8"
services:
  davos:
    image: linuxserver/davos
    container_name: davos
    restart: unless-stopped
    ports:
      #  This is the default port that davos runs under
      - 8080:8080
    volumes:
      #  davos's config location. This is where it stores its database file and logs.
      - < /path/to/your/config>:/config
      #  davos's file download location
      - < /path/to/your/download>:/download
    environment:
      #  User ID
      - PUID=100
      #  Group ID
      - PGID=100
