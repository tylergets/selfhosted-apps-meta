doublecommander

[Double Commander]({{ project_url }}) is a free cross platform open source file manager with two panels side by side. It is inspired by Total Commander and features some new ideas.

Setup

The application can be accessed at:

  • http://yourhost:3000/

By default the user/pass is abc/abc, if you change your password or want to login manually to the GUI session for any reason use the following link:

  • http://yourhost:3000/?login=true
Docker Compose
# doublecommander - Generated by SH-Apps-Meta

version: "3.8"
services:
  doublecommander:
    image: linuxserver/doublecommander
    container_name: doublecommander
    restart: unless-stopped
    ports:
      #  Double Commander desktop gui.
      - 3000:3000
    volumes:
      #  Users home directory in the container, stores program settings.
      - < /path/to/your/config>:/config
      #  Host data directories, mount as many as needed.
      - < /path/to/your/data>:/data
    environment:
      #  User ID
      - PUID=100
      #  Group ID
      - PGID=100
      #  Specify a timezone to use EG Europe/London.
      - TZ=Europe/London
