bazarr

[{{ project_name|capitalize }}]({{ project_url }}) is a companion application to Sonarr and Radarr. It can manage and download subtitles based on your requirements. You define your preferences by TV show or movie and Bazarr takes care of everything for you.

docs
Setup
  • Once running the URL will be http://<host-ip>:6767.
  • You must complete all the setup parameters in the webui before you can save the config.
Docker Compose
# bazarr - Generated by SH-Apps-Meta

version: "3.8"
services:
  bazarr:
    image: linuxserver/bazarr
    container_name: bazarr
    restart: unless-stopped
    ports:
      #  Allows HTTP access to the internal webserver.
      - 6767:6767
    volumes:
      #  Location of your movies
      - < /path/to/your/movies>:/movies
      #  Location of your TV Shows
      - < /path/to/your/tv>:/tv
      #  Bazarr data
      - < /path/to/your/config>:/config
    environment:
      #  User ID
      - PUID=100
      #  Group ID
      - PGID=100
      #  Specify a timezone to use EG Europe/London
      - TZ=Europe/London
