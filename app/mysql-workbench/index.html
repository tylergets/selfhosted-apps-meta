mysql-workbench

[MySQL Workbench]({{ project_url }}) is a unified visual tool for database architects, developers, and DBAs. MySQL Workbench provides data modeling, SQL development, and comprehensive administration tools for server configuration, user administration, backup, and much more.

docker
docs
Setup

The application can be accessed at:

  • http://yourhost:3000/

By default the user/pass is abc/abc, if you change your password or want to login manually to the GUI session for any reason use the following link:

  • http://yourhost:3000/?login=true
Docker Compose
# mysql-workbench - Generated by SH-Apps-Meta

version: "3.8"
services:
  mysql-workbench:
    image: linuxserver/mysql-workbench
    container_name: mysql-workbench
    restart: unless-stopped
    ports:
      #  Mysql Workbench desktop gui.
      - 3000:3000
    volumes:
      #  Users home directory in the container, stores program settings.
      - < /path/to/your/config>:/config
    environment:
      #  User ID
      - PUID=100
      #  Group ID
      - PGID=100
      #  Specify a timezone to use EG Europe/London.
      - TZ=Europe/London
