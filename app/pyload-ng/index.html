pyload-ng

[pyLoad]({{ project_url }}) is a Free and Open Source download manager written in Python and designed to be extremely lightweight, easily extensible and fully manageable via web.

Setup

Access the web interface at http://your-ip:8000 the default login is: username - pyload password - pyload

For general usage please see the pyLoad wiki here .

Docker Compose
# pyload-ng - Generated by SH-Apps-Meta

version: "3.8"
services:
  pyload-ng:
    image: linuxserver/pyload-ng
    container_name: pyload-ng
    restart: unless-stopped
    ports:
      #  Allows HTTP access to the application
      - 8000:8000
    volumes:
      #  pyLoad Configuration and files database
      - < /path/to/your/config>:/config
      #  Destination of pyLoad downloads
      - < /path/to/your/downloads>:/downloads
    environment:
      #  User ID
      - PUID=100
      #  Group ID
      - PGID=100
      #  Specify a timezone to use EG Europe/London
      - TZ=Europe/London
