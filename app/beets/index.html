beets

[{{ project_name|capitalize }}]({{ project_url }}) is a music library manager and not, for the most part, a music player. It does include a simple player plugin and an experimental Web-based player, but it generally leaves actual sound-reproduction to specialized tools.

docker
docs
Setup

Edit the config file in /config

To edit the config from within the container use beet config -e

For a command prompt as user abc docker exec -it -u abc beets bash

See Beets for more info.

Contains beets-extrafiles plugin, configuration details

Docker Compose
# beets - Generated by SH-Apps-Meta

version: "3.8"
services:
  beets:
    image: linuxserver/beets
    container_name: beets
    restart: unless-stopped
    ports:
      #  Application WebUI
      - 8337:8337
    volumes:
      #  Configuration files.
      - < /path/to/your/config>:/config
      #  Music library
      - < /path/to/your/music>:/music
      #  Non processed music
      - < /path/to/your/downloads>:/downloads
    environment:
      #  User ID
      - PUID=100
      #  Group ID
      - PGID=100
      #  Specify a timezone to use EG Europe/London.
      - TZ=Europe/London
