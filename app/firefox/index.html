[Firefox]({{ project_url }}) Browser, also known as Mozilla Firefox or simply Firefox, is a free and open-source web browser developed by the Mozilla Foundation and its subsidiary, the Mozilla Corporation. Firefox uses the Gecko layout engine to render web pages, which implements current and anticipated web standards.

Setup

The application can be accessed at:

  • http://yourhost:3000/

By default the user/pass is abc/abc, if you change your password or want to login manually to the GUI session for any reason use the following link:

  • http://yourhost:3000/?login=true
Docker Compose
# firefox - Generated by SH-Apps-Meta

version: "3.8"
services:
  firefox:
    image: linuxserver/firefox
    container_name: firefox
    restart: unless-stopped
    ports:
      #  Firefox desktop gui.
      - 3000:3000
    volumes:
      #  Users home directory in the container, stores local files and settings
      - < /path/to/your/config>:/config
    environment:
      #  User ID
      - PUID=100
      #  Group ID
      - PGID=100
      #  Specify a timezone to use EG Europe/London.
      - TZ=Europe/London
