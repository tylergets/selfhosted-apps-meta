Setup

The webui is at <your ip>:8200 , create backup jobs etc via the webui, for local backups select /backups as the destination. For more information see Duplicati.

Docker Compose
# duplicati - Generated by SH-Apps-Meta

version: "3.8"
services:
  duplicati:
    image: linuxserver/duplicati
    container_name: duplicati
    restart: unless-stopped
    ports:
      #  http gui
      - 8200:8200
    volumes:
      #  Contains all relevant configuration files.
      - < /path/to/your/config>:/config
      #  Path to store local backups.
      - < /path/to/your/backups>:/backups
      #  Path to source for files to backup.
      - < /path/to/your/source>:/source
    environment:
      #  User ID
      - PUID=100
      #  Group ID
      - PGID=100
      #  Specify a timezone to use EG Europe/London
      - TZ=Europe/London
      #  Optionally specify any [CLI variables](https://duplicati.readthedocs.io/en/latest/07-other-command-line-utilities/) you want to launch the app with
      - CLI_ARGS=
