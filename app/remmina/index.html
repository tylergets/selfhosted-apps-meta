remmina

[Remmina]({{ project_url }}) is a remote desktop client written in GTK, aiming to be useful for system administrators and travellers, who need to work with lots of remote computers in front of either large or tiny screens. Remmina supports multiple network protocols, in an integrated and consistent user interface. Currently RDP, VNC, SPICE, NX, XDMCP, SSH and EXEC are supported.

docker
docs
Setup

The application can be accessed at:

  • http://yourhost:3000/

By default the user/pass is abc/abc, if you change your password or want to login manually to the GUI session for any reason use the following link:

  • http://yourhost:3000/?login=true

# remmina - Generated by SH-Apps-Meta

version: "3.8"
services:
  remmina:
    image: linuxserver/remmina
    container_name: remmina
    restart: unless-stopped
    ports:
      #  Remmina desktop gui.
      - 3000:3000
    volumes:
      #  Users home directory in the container, stores program settings.
      - < /path/to/your/config>:/config
    environment:
      #  User ID
      - PUID=100
      #  Group ID
      - PGID=100
      #  Specify a timezone to use EG Europe/London.
      - TZ=Europe/London
