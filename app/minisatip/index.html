minisatip

[{{ project_name|capitalize }}]({{ project_url }}) is a multi-threaded satip server version 1.2 that runs under Linux and it was tested with DVB-S, DVB-S2, DVB-T, DVB-T2, DVB-C, DVB-C2, ATSC and ISDB-T cards.

docker
docs
Setup

Best used in conjunction with tvheadend

There is no setup per se, other than adding your cards for passthrough.

You can then use your cards as DVB inputs in apps such as tvheadend.

Docker Compose
# minisatip - Generated by SH-Apps-Meta

version: "3.8"
services:
  minisatip:
    image: linuxserver/minisatip
    container_name: minisatip
    restart: unless-stopped
    ports:
      #  Status Page WebUI
      - 8875:8875
      #  RTSP Port
      - 554:554
      #  App Discovery
      - 1900/udp:1900/udp
    volumes:
      #  Configuration files.
      - < /path/to/your/config>:/config
      #  Configuration files and minisatip data
      - < /path/to/your/config>:/config
    environment:
      #  User ID
      - PUID=100
      #  Group ID
      - PGID=100
      #  Specify a timezone to use EG Europe/London.
      - TZ=Europe/London
      #  Specify specific run params for minisatip
      - RUN_OPTS=
      #  Supported values are LATEST, PLEXPASS or a specific version number.
      - VERSION=latest
