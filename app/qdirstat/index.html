[QDirStat]({{ project_url }}) Qt-based directory statistics: KDirStat without any KDE -- from the author of the original KDirStat.

docker
docs
Setup

The application can be accessed at:

  • http://yourhost:3000/

By default the user/pass is abc/abc, if you change your password or want to login manually to the GUI session for any reason use the following link:

  • http://yourhost:3000/?login=true
Docker Compose
# qdirstat - Generated by SH-Apps-Meta

version: "3.8"
services:
  qdirstat:
    image: linuxserver/qdirstat
    container_name: qdirstat
    restart: unless-stopped
    ports:
      #  QdirStat desktop gui.
      - 3000:3000
    volumes:
      #  Users home directory in the container, stores qdirstat settings and scans.
      - < /path/to/your/config>:/config
      #  Data you want to analyze disk usage information of.
      - < /path/to/your/data>:/data
    environment:
      #  User ID
      - PUID=100
      #  Group ID
      - PGID=100
      #  Specify a timezone to use EG Europe/London.
      - TZ=Europe/London
