qbittorrent

The [{{ project_name|capitalize }}]({{ project_url }}) project aims to provide an open-source software alternative to µTorrent. qBittorrent is based on the Qt toolkit and libtorrent-rasterbar library.

docker
docs
Setup

The webui is at <your-ip>:8080 and the default username/password is admin/adminadmin.

Change username/password via the webui in the webui section of settings.

WEBUI_PORT variable

Due to issues with CSRF and port mapping, should you require to alter the port for the webui you need to change both sides of the -p 8080 switch AND set the WEBUI_PORT variable to the new port.

For example, to set the port to 8090 you need to set -p 8090:8090 and -e WEBUI_PORT=8090

This should alleviate the "white screen" issue.

If you have no webui , check the file /config/qBittorrent/qBittorrent.conf

edit or add the following lines

WebUI\Address=*

WebUI\ServerDomains=*

If you are running a very old (3.x) kernel you may run into this issue which can be worked around using this method

# qbittorrent - Generated by SH-Apps-Meta

version: "3.8"
services:
  qbittorrent:
    image: linuxserver/qbittorrent
    container_name: qbittorrent
    restart: unless-stopped
    ports:
      #  WebUI
      - 8080:8080
      #  tcp connection port
      - 6881:6881
      #  udp connection port
      - 6881/udp:6881/udp
    volumes:
      #  Contains all relevant configuration files.
      - < /path/to/your/config>:/config
      #  Location of downloads on disk.
      - < /path/to/your/downloads>:/downloads
    environment:
      #  User ID
      - PUID=100
      #  Group ID
      - PGID=100
      #  Specify a timezone to use EG Europe/London
      - TZ=Europe/London
      #  for changing the port of the webui, see below for explanation
      - WEBUI_PORT=8080
